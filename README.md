# Reserve Transport System

## Description
The **Reserve Transport System** is a web-based application that allows users to view available transport vehicles and make reservations online. Users can select a vehicle, enter personal details, and confirm bookings. The system keeps track of all successful bookings.

## Features
- View available vehicles (Bus, Cab, Train, Auto, Flight)
- Dynamic booking form for Name, Email, Phone, and Date
- Successful bookings displayed in real-time
- Responsive and interactive UI
- Backend API integration with Node.js, Express, and MongoDB

## Technologies Used
- Frontend: React.js, HTML, CSS
- Backend: Node.js, Express.js
- Database: MongoDB

## How It Works
1. Users select a vehicle from the list.
2. Fill in Name, Email, Phone, and Booking Date.
3. Submit the form to save booking in the database.
4. View all successful bookings dynamically.
